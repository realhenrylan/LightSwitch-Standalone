请帮我用以下步骤安装 LightSwitch Standalone 到我的电脑：

## 任务
从 GitHub Release 下载 LightSwitch.exe 到桌面并确保可以运行。

## 步骤

### 第 1 步：暂时禁用 Windows Defender 实时保护（否则会拦截未签名的 exe）
用 PowerShell 执行以下命令：
```powershell
# 临时关闭实时监控（不影响其他安全功能）
Set-MpPreference -DisableRealtimeMonitoring $true
# 确认已关闭
Get-MpPreference | Select-Object DisableRealtimeMonitoring
```

### 第 2 步：下载 LightSwitch.exe
从 GitHub Releases 下载到桌面：
```powershell
# 下载文件到桌面
$url = "https://github.com/realhenrylan/LightSwitch-Standalone/releases/download/v1.0.0/LightSwitch.exe"
$dest = "$env:USERPROFILE\Desktop\LightSwitch.exe"
Invoke-WebRequest -Uri $url -OutFile $dest
# 验证下载成功
Get-Item $dest | Select-Object Name, Length
```

### 第 3 步：重新启用 Windows Defender 实时保护
```powershell
# 重新开启实时监控
Set-MpPreference -DisableRealtimeMonitoring $false
# 确认已开启
Get-MpPreference | Select-Object DisableRealtimeMonitoring
```

### 第 4 步：验证安装
- 确认桌面上有 LightSwitch.exe
- 用户可以双击运行
- 提示：第一次运行时 Windows SmartScreen 仍可能弹警告，点击「仍要运行」即可。运行几次后信誉建立，警告自动消失。

## 注意事项
- 此操作仅适用于我自己的合法软件（开源项目 LightSwitch Standalone）
- 临时关闭 Defender 仅用于下载安装阶段，完成后立即重新开启
- 软件源码地址：https://github.com/realhenrylan/LightSwitch-Standalone
- 如果后续 Windows Defender 仍报毒，请提交误报到 https://www.microsoft.com/en-us/wdsi/filesubmission
